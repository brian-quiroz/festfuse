# FestFuse

Festival planning app that helps users discover artists, 
resolve schedule conflicts, and build their perfect festival experience.

## Stack
- Frontend: Next.js + React + TypeScript + Tailwind
- Backend: FastAPI + Python (coming soon)
- Database: PostgreSQL (coming soon)

## Design
- Dark mode first
- Primary color: Cyan #00E5FF
- Secondary color: Yellow #E8FF47
- Base background: #110D24
- Font: Plus Jakarta Sans
- Component style: clean, minimal, no excessive shadows or gradients

## Build order
1. Artist page
2. Explore page
3. Quick Picks
4. Compare page
5. Schedule page
